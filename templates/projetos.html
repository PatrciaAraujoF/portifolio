Projeto Django para Gestão de Fábrica de Rações

Desenvolvi um sistema para a gestão de fábricas de rações, utilizando o framework Django. Este sistema abrange desde o cadastro e recebimento de matérias-primas, o controle detalhado da produção de rações e expedição. Uma característica marcante é o sistema de rastreabilidade robusto, que permite rastrear cada lote de ração, documentando os lotes individuais de cada ingrediente utilizado, garantindo segurança e qualidade na produção. O deploy foi realizado utilizando Nginx e Gunicorn para assegurar performance e escalabilidade.
