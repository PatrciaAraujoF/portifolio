E-Commerce para Venda de Roupas Femininas



Criei uma plataforma de e-commerce avançada voltada para a comercialização de roupas femininas, incorporando de maneira eficaz técnicas de SEO e estratégias de marketing digital para aumentar a visibilidade e o engajamento do público.

Este projeto foi desenvolvido desde a criação da logomarca até a definição completa da paleta de cores.

O deploy foi realizado na AWS, utilizando Nginx e Gunicorn, o que assegura alta performance e escalabilidade conforme necessário.



Loja Virtual

A loja virtual exibe um design elegante e feminino, com um template que destaca a beleza e a sofisticação. O site inclui funcionalidades essenciais como sistema de cadastro, carrinho de compras e categorizações bem definidas para cada tipo de produto.





Destaques

A loja conta com um sistema de destaques individualizado para cada categoria, proporcionando uma navegação intuitiva e fácil para os usuários.
