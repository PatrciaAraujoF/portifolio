Projeto Django para Controle de Estoque de Peças para Mecânicos



Desenvolvi um sistema para gestão de peças destinado a mecânicos, utilizando a poderosa estrutura do Django. Este sistema foi cuidadosamente projetado para um ambiente familiar e acolhedor, onde a simplicidade é primordial, atendendo de maneira eficaz às demandas de um pequeno negócio. Em um contexto de acesso restrito, eliminamos a necessidade de um sistema de login. A implantação foi realizada com Nginx e Gunicorn, assegurando rapidez e eficiência no desempenho.
